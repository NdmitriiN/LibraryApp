# Приложение для учёта выданных книг в библиотеке

## Описание
Курсовая работа по разработке Windows Forms приложения для автоматизации учёта выданных книг в библиотеке.

Приложение позволяет вести учёт книг, читателей и операций по выдаче/возврату литературы.

## Основные возможности
- Ведение каталога книг (добавление, редактирование, удаление, поиск)
- Ведение базы читателей
- Оформление выдачи книг (с проверкой наличия свободных экземпляров)
- Оформление возврата книг
- Просмотр текущих выдач с подсветкой просроченных
- Автоматическое обновление количества доступных экземпляров

## Используемые технологии
- C# 
- Windows Forms (.NET 8)
- Entity Framework Core
- SQL Server LocalDB

## Структура проекта
- **Models** — модели данных (Book, Reader, Loan)
- **Data** — работа с базой данных (DbContext, инициализация)
- **Forms** — формы приложения (справочники, выдача, возврат)
- **MainForm** — главное окно программы

## Запуск приложения
1. Откройте решение в Visual Studio 2022 или выше.
2. Убедитесь, что установлен SQL Server LocalDB.
3. Запустите проект (F5).
4. При первом запуске база данных создастся автоматически с тестовыми данными.

## База данных
Используется локальная база SQL Server LocalDB.  
Строка подключения находится в файле `Data/LibraryDbContext.cs`.

При первом запуске приложение автоматически создаёт таблицы и добавляет несколько тестовых книг и читателей.
